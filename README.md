AWSM (AWS CLI Manager)

A CLI tool to manage AWS resources sanely.

## Installation

```sh
git clone https://github.com/thesobercoder/awsm.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/awsm
```

## Usage

```sh
awsm
```
